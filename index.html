Explain it like I am 10

Think of this app like a smart office. The Router Agent is the front desk. It listens to your question and sends you to the right room.

  • GTM Room: answers product questions and pricing.
  • Outreach Room: writes content, finds leads, and can send emails if you clearly ask.

All rooms share one backpack called AgentState with question, context, answer, and step logs in steps.

normal edge dashed = conditional route

Simple version: Router picks GTM or Outreach, then each path has rule checks before END.

GTM path

  • gtm_retrieve pulls context from KB + web tools.
  • pricing_gate detects pricing intent.
  • If pricing, collect_email enforces verified email.
  • gtm_generate writes final answer from context.

Outreach path

  • outreach_research uses Apollo for lead tasks, else KB + web.
  • outreach_generate drafts post/email content.
  • send_gate decides send vs review.
  • outreach_send sends via SendGrid when explicitly requested.
